# CSVデータの集計

CSVデータを集計し、参加者ごとのスコアを自動的に計算するプログラムです。

## 機能
- CSVファイルの読み込み
- 参加者ごとのスコア集計
- 平均値・最大値・最小値の計算

## 使用技術
- Python
